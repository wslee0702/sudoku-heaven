# 수도쿠 프로젝트 기억 파일

## 파트너십 원칙 (가장 중요)
- 피드백은 반드시 기록하고 다음 작업에 즉시 반영
- 대화가 쌓일수록 효율이 높아지는 것이 목표
- 사용자는 코딩 초보 → 항상 쉬운 언어로 설명
- 결정이 필요할 때는 선택지 제시 후 기다림
- 코딩은 전부 대신 해줌

## 피드백 및 학습 사항
- 사용자가 "기억하고 발전시켜 달라"고 명시적으로 요청 → MEMORY.md 적극 활용
- 결정 필요 시: AskUserQuestion 도구 사용해 선택지 제시
- 기능 추가 전 EnterPlanMode → 계획 승인 후 구현
- 구현 범위가 클 때는 TodoWrite로 진행 상황 추적

## 프로젝트 개요
- 사용자: 코딩 완전 초보, 바이브코딩으로 수도쿠 게임 제작
- 언어: 한국어로 소통, 쉬운 설명 필수
- GitHub 가입 완료, Supabase 가입 완료 (프로젝트 생성 완료)
- 온라인 배포는 기능 추가 후 진행 예정

## 구현된 기능 (v3 - 3차 업데이트)
**기본 기능:**
- 10단계 난이도 슬라이더 (입문 Lv.1-2 / 쉬움 Lv.3-4 / 보통 Lv.5-6 / 어려움 Lv.7-8 / 전문가 Lv.9-10)
- 타이머, 일시정지/재개 (P키, 스페이스)
- 메모 모드 (M키)
- 키보드 조작 (방향키, 1-9, Delete/0, M, H, P)
- 힌트 5개 (각 60초 패널티)
- 오답 표시 토글 (기본 ON)
- 새 게임 / 다시하기
- 완성 시 컨페티 + 축하 화면

**v3 신규 기능:**
- 타이틀 "Sudoku Heaven" 그라디언트 스타일 (보라→핑크→황금)
- 레벨 1-10으로 축소 (getDifficultyInfo, getClueCount 모두 1-10 기준)
- 리더보드 필터: 전체 + 레벨별 (레벨 1~10 개별 선택 가능)

**v2 신규 기능:**
- Undo/Redo (Ctrl+Z, Ctrl+Y, 헤더 ↩↪ 버튼)
- 줄/박스 완성 시 초록 번쩍임 (::after 오버레이 방식)
- 전체 완성 시 골드 번쩍임
- 키패드 숫자 선택 하이라이트 (같은 숫자 파란색 강조)
- 셀 클릭 시 selectedNum 자동 입력
- Enter 키 = 1→2→...→9→0→1 순환 + 자동 입력
- 레벨 선택 첫 화면 (5개 티어 버튼 + 세부 슬라이더)
- 시즌 리더보드 + 자동 월별 시즌 종료
- 명예의 전당 (과거 시즌 아코디언)
- 기존 localStorage 데이터 자동 마이그레이션

## localStorage 구조 (v2)
- `sudoku_season_meta`: 현재 시즌 정보 { id, name, startDate, autoEndDate }
- `sudoku_records_S{id}`: 해당 시즌 기록 배열 (최대 200개)
- `sudoku_archive`: 종료된 시즌 메타 배열
